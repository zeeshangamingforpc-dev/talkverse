# TalkVerse

TalkVerse is a web-based AI text-to-speech (TTS) platform built with modern HTML, CSS, and JavaScript. Users can generate audio from text using the ElevenLabs API and manage multiple voices, settings, and history. The platform is fully responsive and supports multiple languages.

---

## Features

- **Text-to-Speech Generation:** Convert your text to high-quality speech.
- **Multiple Voices:** Choose from a variety of AI voices.
- **Voice Selection:** Select and preview voices before generating.
- **Audio Player:** Play generated audio directly in the browser.
- **History:** Keep track of generated audio clips.
- **Progress Indicators:** Real-time progress bars for audio generation.
- **Responsive Design:** Works on desktop, tablet, and mobile.
- **Themes & Text Sizes:** Multiple themes and text-size options.
- **Toast Notifications & Loading Overlays:** Real-time feedback while processing.
